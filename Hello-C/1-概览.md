#### S1 概览
```
0.1972  Dennis Ritchie  贝尔实验室
1.控制特性、高效性、可移植性、灵活性
2.面向对象编程（一种哲学思想：让语言来适应问题，而不是让问题适应语言）工具
3.使用C语言的七个步骤：
 （1）定义程序目标：用一般概念来考虑问题
 （2）设计程序
 （3）编写代码
 （4）编译
 （5）运行程序
 （6）测试和调试程序
 （7）维护和修改程序
4.                            库例程代码
                                 ↓                                  
  源代码→编译器→目标代码（.obj）→链接器→可执行代码（.exe）
                                 ↑
                              启动代码（start-up code）
5.问题：显示程序执行的窗口在程序终止时突然消失
  解决：getchar()使程序暂停，直到按下[Enter]，放return之前
       有时根据程序需要可能已经有一个等待按键的指令，这种情况需要两次getchar()
6.保持C精神：（1）相信程序员
            （2）不妨碍程序员做需要完成的事
            （3）让语言保持短小简单
            （4）只提供一种方法来执行一个操作
            （5）使程序运行速度快，即使不能保证其可移植性
```
